# Contributing

**benpocket** is a personal project built with friends and still in its very early
stages, so it's **not open to code contributions** (pull requests) right now.

## Feature requests & bug reports

These are very welcome! Please [open an issue](https://github.com/invisal/craftbox/issues)
and include:

- **Bug reports** — what you did, what you expected, what happened instead, and your
  OS (Windows/macOS/Linux). Screenshots or screen recordings help a lot.
- **Feature requests** — what problem you're trying to solve and which tool it
  relates to (File Explorer, HTTP Client, Kubernetes, Screen Capture, Screen
  Recorder), or if it's a new tool idea.

Before opening an issue, please search existing issues to avoid duplicates.
