# benpocket

**benpocket** is a personal multi-purpose toolkit — a Swiss Army knife of small tools we
find ourselves reaching for every day, combined into one desktop app instead of
scattered across a dozen different apps and websites.

**This project is in very early stages.** Expect rough edges, missing features, and
things that break.

## Tools

- **File Explorer** — browse, edit, and manage local files (and R2/S3, in progress)
- **HTTP Client** — Postman-style tool for building and sending HTTP/WebSocket requests
- **Kubernetes** — GUI client for browsing cluster resources (pods, deployments, services, etc.)
- **Screen Capture** — take screenshots
- **Screen Recorder** — record, edit, and export screen recordings

## Development

```bash
npm install
npm run dev
```

### Build

```bash
npm run build:win     # Windows
npm run build:mac     # macOS
npm run build:linux   # Linux
```
