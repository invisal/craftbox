# Security Policy

**benpocket** is a personal project in early development, run locally and not deployed
as a service, so the attack surface is small — but if you find a security issue
(e.g. something that could leak local files, credentials, or cluster/API tokens
handled by one of the tools), please report it privately rather than opening a
public issue.

## Reporting a vulnerability

Email **report@benpocket.com** with:

- A description of the issue and its potential impact
- Steps to reproduce
- Which tool is affected (File Explorer, HTTP Client, Kubernetes, Screen Capture,
  Screen Recorder)

We'll do our best to respond and fix confirmed issues promptly, given this is a
side project.
